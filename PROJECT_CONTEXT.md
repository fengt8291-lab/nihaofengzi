# 四、主要项目展示


## 1. DailyTasks

项目类型：

现实连接型数字产品。


核心理念：

帮助年轻人在高度线上化环境中重新连接现实生活。


解决的问题：

部分年轻人在信息过载、线上娱乐和社交比较中容易产生低连接状态。


产品方向：

通过轻量任务，引导用户：

- 回到现实
- 体验生活
- 建立积极反馈



## 2. 今夜补救

项目类型：

睡眠恢复辅助产品。


核心理念：

帮助用户从作息失控状态逐步恢复。


关注：

- 睡眠管理
- 行为调整
- 个性化建议



## 3. AI Soul

项目类型：

AI实验项目。


方向：

探索AI工具、人机交互以及个人生产力提升。



## 4. AI员工与一人公司研究

研究方向：

生成式AI和AI Agent进入个人工作流程后，对传统管理方式的影响。


关注：

- AI作为数字劳动力
- 人类与AI协作
- 未来个人组织模式
